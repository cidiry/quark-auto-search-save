# 夸克网盘自动转存

**夸克网盘自动转存** 是一个基于 Python 和 NiceGUI 构建的本地化轻量级工具。它提供了一个现代化的 Web 界面，帮助用户快速搜索夸克网盘资源，并一键将其转存到自己的网盘中。

## ✨ 功能特性

- **🔍 聚合搜索**：内置第三方搜索接口，快速查找全网夸克网盘分享资源。
- **💾 一键转存**：无需跳转，直接在工具内将搜索到的资源保存到你的指定文件夹。
- **🍪 账号管理**：支持 Cookie 验证与状态保持，实时显示昵称与连接状态。
- **⚡ 异步处理**：搜索与转存操作异步执行，界面流畅不卡顿。
- **🖥️ 简洁 UI**：基于 Material Design 风格的响应式界面。

## 🛠️ 安装与运行

### 1. 环境准备
确保你的电脑已安装 Python 3.8 或更高版本。

### 2. 克隆项目
```bash
git clone https://github.com/你的用户名/QuarkHunter.git
cd QuarkHunter
```

### 3. 安装依赖
建议使用虚拟环境（可选），然后安装依赖包：
```bash
pip install -r requirements.txt
```

### 4. 启动程序
```bash
python main.py
```
启动后，浏览器会自动打开 http://localhost:8080


## 📖 使用指南
### 第一步：获取 Cookie
为了操作你的网盘，程序需要你的夸克 Cookie。

- 1.在 PC 浏览器中访问并登录 夸克网盘网页版。
- 2.按 F12 打开开发者工具，点击 Network (网络) 标签。
- 3.刷新页面，在列表中找到 sort 或 info 等请求。
- 4.点击该请求，在 Headers (标头) -> Request Headers (请求头) 中找到 Cookie 字段。
- 5.复制整个 Cookie 字符串。

### 第二步：配置与搜索
- 1.将 Cookie 粘贴到 夸克网盘自动转存 界面的配置框中，点击 "验证并保存"。
- 2.在搜索框输入关键词（如 "Python教程"），点击搜索。
- 3.在结果列表中点击 "一键转存" 即可。
